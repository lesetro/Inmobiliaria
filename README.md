# Inmobiliaria - App Móvil para Propietarios

App Android para los propietarios de una inmobiliaria. Permite gestionar su perfil, inmuebles, contratos y pagos.

- Java
- Android Studio
- Retrofit + Gson (consumo de API REST)
- MVVM con LiveData

## Cómo correr

1. Clonar el repo y abrirlo con Android Studio.
2. Esperar a que Gradle sincronice.
3. Ejecutar en un emulador o dispositivo (mínimo Android 7.0).
